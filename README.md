### Backend part for starplatinum-app

## Installation

```bash
$ yarn
# run docker desktop
$ docker-compose up
$ yarn prisma migrate dev
```

## Running the server

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Technologies

- NestJS
- Yarn
- Typescript
- Prisma
- ESLint & Prettier
- Bcrypt
- JWT
- Class Transformer
- Multer
- Nodemailer

_Find more information in the `package.json` file_
