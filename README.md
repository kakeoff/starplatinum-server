### Backend part for starplatinum-app

## Installation

```bash
$ yarn
# run docker desktop
$ docker-compose up
$ yarn prisma migrate dev
```

## Running the server

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
